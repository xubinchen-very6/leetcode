### 从排序数组中删除重复的项

__给定一个有序数组，你需要原地删除其中的重复内容，使每个元素只出现一次,并返回新的长度__
__不要另外定义一个数组，您必须通过用_O(1)_额外内存原地修改输入的数组来做到这一点__



> 给定数组: nums = [1,1,2],
你的函数应该返回新长度 2, 并且原数组nums的前两个元素必须是1和2
不需要理会新的数组长度后面的元素





```
 1. done = False
 2. i = 0
 3. while not done:
 4.     if i<=len(nums)-1:
 5.         if nums[i] in nums[:i]:
 6.             nums.pop(i)
 7.             i-=1
 8.         i+=1
 9.     else:
10.         done=True
11. print(nums)
```

    判断完毕指针+1
